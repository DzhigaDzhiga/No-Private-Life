## Views
### View for the teacher table:
```sql
CREATE VIEW teacher_view AS
SELECT id, name, email
FROM teacher;
```
### View for the subject table:
```sql
CREATE VIEW subject_view AS
SELECT id, name
FROM subject;
```
### View for the subject_teacher table:
```sql
CREATE VIEW subject_teacher_view AS
SELECT s.name AS subject_name, t.name AS teacher_name
FROM subject_teacher st
INNER JOIN subject s ON st.subject_id = s.id
INNER JOIN teacher t ON st.teacher_id = t.id;
```
### View for the faculty table:
```sql
CREATE VIEW faculty_view AS
SELECT id, name
FROM faculty;
```
### View for the courses table:
```sql
CREATE VIEW courses_view AS
SELECT c.id, c.name, f.name AS faculty_name
FROM courses c
INNER JOIN faculty f ON c.faculty_id = f.id;
```
### View for the group table:
```sql
CREATE VIEW group_view AS
SELECT g.id, g.name, c.name AS course_name
FROM group g
INNER JOIN courses c ON g.course_id = c.id;
```
### View for the student table:
```sql
CREATE VIEW student_view AS
SELECT s.id, s.name, s.email, g.name AS group_name
FROM student s
INNER JOIN group g ON s.group_id = g.id;
```
### View for the marks table:
```sql
CREATE VIEW marks_view AS
SELECT m.id, s.name AS student_name, su.name AS subject_name, m.mark
FROM marks m
INNER JOIN student s ON m.student_id = s.id
INNER JOIN subject su ON m.subject_id = su.id;
```
## Triggers
### Trigger: When a new teacher is added to the teacher table, insert a corresponding row in the subject_teacher table for each subject.
```sql
CREATE TRIGGER insert_subject_teacher
AFTER INSERT ON teacher
FOR EACH ROW
BEGIN
    INSERT INTO subject_teacher (subject_id, teacher_id)
    SELECT id, NEW.id
    FROM subject;
END;
```
### Trigger: When a student is added to the student table, update the group's student count in the group table.
```sql
CREATE TRIGGER update_group_student_count
AFTER INSERT ON student
FOR EACH ROW
BEGIN
    UPDATE `group`
    SET student_count = student_count + 1
    WHERE id = NEW.group_id;
END;
```
### Trigger: When a mark is added to the marks table, calculate the average mark for that student and update it in the student table.
```sql
CREATE TRIGGER update_student_average_mark
AFTER INSERT ON marks
FOR EACH ROW
BEGIN
    UPDATE student
    SET average_mark = (
        SELECT AVG(mark)
        FROM marks
        WHERE student_id = NEW.student_id
    )
    WHERE id = NEW.student_id;
END;
```
### Trigger: When a faculty is deleted from the faculty table, cascade delete the corresponding courses and groups associated with that faculty.
```sql 
CREATE TRIGGER cascade_delete_courses_and_groups
AFTER DELETE ON faculty
FOR EACH ROW
BEGIN
    DELETE FROM courses
    WHERE faculty_id = OLD.id;
    DELETE FROM `group`
    WHERE course_id IN (
        SELECT id
        FROM courses
        WHERE faculty_id = OLD.id
    );
END;
```
